A real Sim.Viz recorder run — a ~440 m block of the synthetic 5×5 km city, weave ON. Peds thread the sidewalks; the amber rings mark any pair within 0.5 m.

What you're seeing. Every ped's position is the engine's deterministic low-power pose — a pure function of its route, seed, the baked sidewalk width, and time. The weave keeps opposing flows on their own half and fans same-direction flows into a band, so the crowd threads the sidewalks with few overlaps despite no neighbour lookups. Reload → identical playback (the same determinism that makes server == image-generator bit-for-bit). See the on/off corridor demo for the weave contrast.
