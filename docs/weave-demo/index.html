What you're seeing. Each ped's lateral position on the sidewalk is a pure, seeded function of its own route arc-length — no neighbour lookups. Opposing directions keep to their own half (a soft keep-right), so head-on flows separate instead of interpenetrating; a same-direction flow fans into a band that fills the sidewalk's own width (wider on the 4 m / 6 m sidewalks). Turn the weave off and both streams collapse onto the centreline — the pass-through count jumps.

Why it matters. Because the pose is a pure function of (route, seed, width, time), a remote image-generator recomputes the identical positions from the once-broadcast route — server == image generator, bit-for-bit, with no per-frame position stream for the low-power crowd. The same determinism you see on reload.
